API Endpoints

GET /api/decks GET

Retrieve all decks

curl http://localhost:5000/api/decks
GET /api/decks/{id} GET

Retrieve a specific deck by ID

curl http://localhost:5000/api/decks/{id}
POST /api/decks POST

Create a new deck

curl -X POST http://localhost:5000/api/decks \ -H "Content-Type: application/json" \ -d '{"name":"My Deck","cards":["Card 1","Card 2"]}'
PUT /api/decks/{id} PUT

Update an existing deck

curl -X PUT http://localhost:5000/api/decks/{id} \ -H "Content-Type: application/json" \ -d '{"id":"{id}","name":"Updated","cards":[]}'
DELETE /api/decks/{id} DELETE

Delete a deck

curl -X DELETE http://localhost:5000/api/decks/{id}
